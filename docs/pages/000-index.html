Introduction to siimple colors
siimple colors is a minimal, ready for use, elegant and organized color system for designing clean and beautiful UI interfaces. It can be integrated in your proyect via 3 different ways:
Pick up a color from our 75 colors to use in your proyect or to starting your own color palette.
Use our css modifiers or our css variables to customize the design of your web applicacion with our color palette.
Our color palette can be imported in your scss project, so you can use only the colors that you really need!
Let's get started
Start with siimple colors adding it to your project using npm or yarn:
## Install using NPM
$ npm install --save @siimple/colors
## Or install using YARN
$ yarn add --save @siimple/colors
Looking for alternative installation methods? Read installation docs.
