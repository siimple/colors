What is siimple-colors?
siimple-colors is a minimal, ready for use, elegant and organized color system for designing clean and beautiful UI interfaces.
You can pick up a color from our color palette to use in your proyect or to starting your own color palette.
Use our css modifiers or our css variables to customize the design of your web applicacion with our color palette.
Our color palette can be imported in your scss project, so you can use only the colors that you really need!
