siimple-colors
An elegant and minimalistic color palette for UI design.
Current version: v{{ version }}
What is siimple-colors?
A color library
A minimal and elegant color palette with 11 colors specially selected to build flat and clean UI designs.
Built in CSS
The library includes CSS variables and selectors for each color in the palette to design a colorful website!
With SASS/SCSS powers
The library also provides SASS/SCSS variables for each color. Just use only the colors that you need!
Installation
You can download siimple-colors using the NPM package manager or using a hosted version in a CDN.
Using NPM
Hosted in a CDN
NPM is the package manager for Node.js. You will need to install Node.js if you don't already have it installed. Open a new terminal window and navigate to your project folder. Then, run the following command:
$ npm install --save siimple-colors
This will install the siimple-colors library files in your project's node_modules folder. Now refer to the main CSS file adding a link element in your HTML pages:
<link rel="stylesheet" href="./node_modules/siimple-colors/css/siimple-colors.min.css">
Just add the following link element in your HTML pages to import the siimple-colors library from the CDN.
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/siimple-colors@{{ version }}/dist/siimple-colors.min.css">
Well done! Now that you have siimple-colors imported in your project, you can start using all colors provided in the siimple-colors library.
